# Hotel Booking

Proyecto para de los servicios relacionados con Reserva de Hotel

---

## Requisitos del Sistema

- ASP.NET 8.0

---

## Configuración del Proyecto

1. Clona este repositorio: `git clone git@github.com:abdielGonzalez/HotelBooking.git`
2. Abre la solución en Visual Studio 2022.
3. Restaura los paquetes NuGet: `nuget restore`
4. Compila el proyecto.

---

## Configuración de la Base de Datos (si aplica)

- Conectar a la BD [CARDS_DB]

## Ejecución del Proyecto

1. Configura el proyecto como proyecto de inicio.
2. Presiona F5 o utiliza el comando "Start" en Visual Studio.
3. Accede a la API a través de la URL predeterminada (por ejemplo, `https://localhost:7241/swagger/index.html`).
